项目 v2.0 版本功能基本完整，处于稳定可运行状态。代码已全部提交至本地 git。

## 会话上下文（本次会话改动）

### 已修复

| 模块 | 问题 | 修复方式 |
|------|------|----------|
| **地图-城市排行** | 排行榜数字与展开记录数不一致 | 新增 `GET /api/map/city-records?city=xxx` 后端 API，展开时调此接口获取该城市**所有聚餐记录**（不限 isCheckin），与排行榜统计口径对齐 |
| **地图-城市详情样式** | 底部弹出面板不符合需求 | 改为 inline 下拉展开式（accordion），点击城市条目下方直接展开打卡记录，保留记录本卡片风格（左侧日期块 + 右侧内容） |
| **记录本-详情页** | 点击记录无法查看详情 | `goDetail` 导航路径从 `/pages/record/detail` → `/subpackages/record/detail`（子包路径） |
| **记录本-详情页卡死** | 加载失败时永久显示"加载中" | 新增 `loadFailed` 状态 + 错误提示 +「重新加载」按钮；增加 `tryLocalFallback` 统一回退逻辑；增加 `g.id` 匹配 |
| **吃什么-标签功能** | 添加菜品无法自定义标签 | 预设标签下方新增自定义标签输入框 +「添加」按钮，已选标签可点击 × 移除 |
| **吃什么-菜品管理** | 无法筛选菜品 | 管理弹窗新增名称搜索框 + 标签筛选按钮（自动提取该菜系所有标签）+「清除」按钮 |
| **我的-子包导航** | 关系图鉴/相册/饭搭子无法打开 | `goRelation/goAlbum/goBuddies` 路径从 `/pages/mine/...` → `/subpackages/mine/...` |
| **我的-移除功能** | 移除了"我的口味"和"导入示例数据"两个功能 | 删除对应 JS 方法、WXML 菜单项和弹窗、WXSS 样式 |

### 已确认

- **地图数据来源**：所有数据来自真实后端 API（MongoDB），无 mock 数据
  - Footprints（城市排行）统计所有聚餐记录 ✓
  - Checkins（打卡记录）仅含 isCheckin:true ✓
  - City-records（城市详情）取所有记录，与排行对齐 ✓
